# Alaska Duck Hunt Environment Reference Pack

This pack contains **36 curated reference images**—three for each of the 12 Alaska habitat areas shown in the supplied menu image. The folders are numbered to match the menu.

Each area includes a mix of broad establishing views, aerial or panoramic terrain references, closer vegetation/ground details, seasonal or weather variation, and habitat-use context where available.

## Included files

- `AREA_MENU_REFERENCE.png` — the supplied area menu.
- `01_...` through `12_...` — one folder per game area, with three images each.
- `INDEX.html` — a browsable visual catalog that works offline.
- `SOURCES_AND_LICENSES.csv` — title, creator, license, source page, and intended reference use for every image.

## Image preparation

The downloaded source images were resized and JPEG-compressed for a practical game-development reference pack. No compositional edits were made beyond resizing, orientation correction, and recompression.

## Licensing and reuse

This is a mixed-license collection. Many images are U.S. federal public-domain works; others require attribution and some use ShareAlike licenses. **Review `SOURCES_AND_LICENSES.csv` before redistributing an image, including it in a shipped game, or creating derivative promotional art.** Attribution and ShareAlike obligations remain attached to the relevant source images.

The images are intended primarily as visual references for environment design, modeling, lighting, texture studies, and concept development.
