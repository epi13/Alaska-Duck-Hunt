# Art Direction

The visual identity is original 16-bit-inspired pixel art: crisp silhouettes, restrained palettes, readable motion, and contemporary lighting. It is an Alaskan nature portrait, not an imitation of any existing hunting game's composition or interface.

Native assets use small pixel grids and integer nearest-neighbor scaling. Birds prioritize identification landmarks over microscopic texture. Environments use four depth bands, asymmetric natural shapes, and location-specific palettes. UI uses a custom block treatment, high-contrast focus frames, and icon-plus-text state communication.

Weather and lighting use palette ramps, dithered fog, sparse particles, reflection bands, and silhouette shifts. CRT, scanlines, curvature, shake, flash, and heavier particles are optional. Reduced-motion and reduced-flash modes replace animated emphasis with static outlines and short opacity changes.

All generated and hand-authored game assets are original project work and distributed under the asset terms stated in the repository license. Source prompts, palettes, and generator inputs must not request imitation of proprietary game assets.
