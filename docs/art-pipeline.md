# Art Pipeline

Asset generation should be deterministic and source-controlled at the recipe level. Palette definitions, pixel primitives, sprite manifests, and atlas metadata are inputs; optimized PNG/WebP atlases and JSON maps are outputs.

Bird sheets share named states: rest, takeoff, wing-up, wing-neutral, wing-down, bank-left, bank-right, climb, dive, hit, fall, escape, distant, and flock. Variant metadata maps sex, age, and seasonal plumage to frames. Habitat generation emits parallax layers, terrain tiles, vegetation, animated water or snow, weather textures, and occluders. UI generation emits reticles, ammunition, buttons, logo treatment, achievements, controller diagrams, guide plates, splash art, and PWA icons.

The second production bird collection replaces the flight-only contract with 16 complete behavior atlases. Each normal species has a 1024×512 atlas containing two 4×4 variants; Sandhill Crane uses 1536×768 for its tall silhouette. Frames are named `<variant>/<pose>/0` in JSON, and every species has a four-frame Field Guide preview. `src/data/bird-sprites.ts` is authoritative. `npm run pack:birds` performs key cleanup, alpha conversion, atlas packing, JSON creation, and preview extraction. The ten superseded flight sheets remain checksummed under `assets/references/current-implemented-bird-sheets`.

The location collection follows the same separation. Twelve original 1280×720 opaque plates live under `public/assets/scenes`; a 512×512 field-retriever sheet provides four rows of animation; and eight 1024×1024 regional habitat atlases provide location-appropriate plant, rock, wood, waterline, and shore occluders. `src/data/scene-art.ts` maps locations to plates, while `src/data/scene-props.ts` owns individually authored map-region-linked prop compositions. Phaser owns only responsive display, animation, and spatial depth ordering. The retained generation sources, keyed masters, prompt records, and reference-license manifests live under `assets/`.

Validation checks dimensions, transparent padding, duplicate hashes, missing states, atlas bounds, nearest-neighbor metadata, file size budgets, and every referenced asset path. Visual review covers silhouette readability at native and TV scale, palette contrast, protected-lookalike differentiation, and phone-safe crops.
