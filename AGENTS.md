# Agent Guide

Preserve the pure deterministic systems in `src/core`, keep content data-driven in `src/data`, and use Phaser only as a presentation adapter. Never introduce copyrighted game assets. Run `npm run check` and `npm run test:e2e` before handoff. Update `CODEX_PROGRESS.md` after substantial phases. Regulatory information requires official citations, a verification date, and the in-game disclaimer.
